# Tableau-Sales-and-Profit-Analysis-Dashboards
Interactive Tableau dashboard analyzing sales and profit performance with dynamic filters, Top N analysis, and geographic insights for data-driven decision making.

## Project Overview
This solution is built using Tableau and focuses on analyzing business performance through two main perspectives:

Sales Performance
Profitability Analysis

The dashboards are designed with a strong emphasis on interactivity, data storytelling, and user experience (UX).

---
## Dashboards Included
1️⃣ Sales Analysis Dashboard

Focuses on:
Sales performance overview
Top-performing products
Monthly sales trends
Geographic distribution of sales
2️⃣ Profit Analysis Dashboard

Focuses on:
Profit distribution
Customer segments analysis
Regional profitability insights

---
## Key Features
🔹 Advanced Interactivity
Dynamic filters (Region, State, etc.)
Start & End Date parameters for time-based analysis
Top N parameter to explore best-performing products/customers
## Smart Map Control
Implemented Show/Hide parameter to toggle map visibility
Enhances usability and reduces dashboard clutter

---
## Data Visualization Techniques
Dual-axis charts (e.g., Profit vs Quantity)
Bar charts & pie charts for comparisons

--- 
## Dashboard Navigation
Seamless navigation buttons between dashboards
Each dashboard tells a distinct story while staying connected
